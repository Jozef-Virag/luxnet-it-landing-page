# LuxNet-it Landing Page

## How to install

1. Clone this repositoy:

```bash
git clone https://github.com/UKF-JozefVirag/luxnet-it-landing-page
```

2. Go into project

```bash
cd luxnet-it-landing-page
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
